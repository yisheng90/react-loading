# `Height`

*number | string - optional*

Defaults to 100%. It is used to set the height of the skeleton. You can set it using `%`, `rem` or `px`.

## Number 
*equivalent to px*

<Preview>
    <Story name="Basic">
        <Skeleton height={30}/>
    </Story>
</Preview>

## Percentage

<Preview>
    <Story name="Percentage">
        <div style={{height: '100px'}}>
            <Skeleton height="50%"/>
        </div>
    </Story>
</Preview>
