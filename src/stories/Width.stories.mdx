# `Width`

*number | string - optional*

Defaults to 100%. It is used to set the width of the skeleton. You can set it using `%`, `rem` or `px`.

## Number 
*equivalent to px*

    <Story name="Basic">
        <Skeleton width={300}/>
    </Story>
</Preview>
